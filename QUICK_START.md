## First Time Setup (Admin Only)

1. **Set the drop channel** (in the channel where you want drops to appear):
   ```
   !setdrop
   ```

2. **Start the drop system**:
   ```
   !startdrops
   ```

That's it! Drops will now appear every 20 seconds in that channel.

## For Players

### Getting Started
```
!start          - Begin your journey
!select nix     - Choose Nix as your starter
!select bruce   - Choose Bruce as your starter  
!select buck    - Choose Buck as your starter
```

### Check Your Progress
```
!profile        - View your stats and characters
!char <name>    - See details about a specific character you own
```

### Opening Crates
```
!crate          - View all crate types
!crate gold     - Open a Gold Crate (100 gems)
!crate emerald  - Open an Emerald Crate (250 gems)
!crate legendary - Open a Legendary Crate (500 gems)
!crate tyrant   - Open a Tyrant Crate (750 gems)
```

### Level Up
```
!levelup        - Use tokens to level up
```

### Catch Drops
When a drop appears, type:
```
!c <code>       - Example: !c tyrant
```

### Trading
```
!t @username    - Start a trade with someone
!offer coins 50 - Offer 50 coins (during trade)
!offer gems 10  - Offer 10 gems (during trade)
!confirm        - Confirm your offer
!cancel         - Cancel the trade
```

## Admin Commands

```
!grant @user tokens 100        - Give 100 tokens to a user
!grant @user coins 500         - Give 500 coins to a user
!grant @user gems 50           - Give 50 gems to a user
!grantchar @user Bali          - Give a specific character to a user
!stopdrops                     - Stop the drop system
```

## Tips

- **Collect all 50+ characters** by opening crates!
- **Level up** by earning tokens from drops and crates
- **Trade** with friends to help each other out
- **Catch drops** quickly - first person to type the code gets the reward!
- Higher tier crates have better character drop rates
